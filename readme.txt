al_mian_xgb.py: Main program for running the active learning framework.
DataProcessor.py: This file is used to select the datasets to be tested.
alstr_XX.py: Classes for all query strategies.
G_Dataset: Contains all synthetic datasets and the dataset generator.
Dataset: Contains all real-world datasets.
xgb_res: Contains the results of the active learning performance of all strategies on all datasets.
Sec6_2_al_main_viz.py + Sec6_2_plot_distribution.py + Sec6_2_concrete_LL4AL_vs_EGAL_viz: Visualizations in Section 6.2 that compare the sample selection process.
Sec_4_1_dataset_analysis.py + Dataset_Analysis_Reports: Scripts and reports for dataset analysis.
